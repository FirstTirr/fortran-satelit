# Satellite Orbit Simulation

This project simulates a satellite orbit around Earth using Fortran for calculation and Python for 3D visualization.

## Structure
- `src/`: Fortran source code
- `viz/`: Python visualization scripts
- `build.ps1`: PowerShell build script

## How to Run
1. Run `./build.ps1` to compile.
2. Run `./exe/orbit_sim.exe` to generate data.
3. Run `python viz/animate_orbit.py` to view animation.
